## ✨ Возможности

- 🥭 150+ уникальных манго и предметов
- 💎 8 уровней редкости (от обычных до космических)
- 🔨 20 рецептов крафта
- 💱 P2P торговля с другими игроками
- 🎁 Ежедневные награды и еженедельные дропы
- ⚡ Энергия, авто-кликеры, критические удары
- 🌐 Глобальный счётчик кликов в реальном времени
- 🎨 Современный тёмный интерфейс

## 🛠️ Технологии

- **Frontend:** HTML5, CSS3, Vanilla JS (ES6 Modules)
- **Backend:** Firebase Authentication + Firestore
- **Хостинг:** GitHub Pages
